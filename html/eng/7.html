Books

If you set a texture as a material and then assign it to an object, the texture is simply "stretched" on the object, taking into account the texture coordinates. But sometimes you need to place certain parts of the texture in certain places on the object.

A texture with calculated coordinates for certain parts of it is called a texture scan.

Now we will learn how to create a texture scan, namely the Unwrap UVW modifier (coordinate scan). We will create a simple book model and protecture it.

now you need to apply the EditMesh modifier(grid editor) to this object.

After that, go to the face selection mode (key 3) and select all the faces of the object (Ctrl + A - select all)

Now, without removing the selection, apply the Unwrap UVW modifier to the object. In the modifier settings that open, click the Edit button to edit the scan and create texture coordinates.

The Edit UVWs window opens for creating a texture scan.

There are different ways to create texture scans. In our case, we have an object in the form of a book, so it is easy to imagine a scan in the form of six photos of the book from different sides, so the Flatten Mapping method is optimal for us. Press Ctrl + A to select all, then select the Select menu and select the Convert vertex to face option in the list that opens. Now we can apply the Flatten Mapping method to our object. To do this, open the mapping -> Flatten Mapping menu.

In the parameters window that opens, click OK. Now you can see the markup for all our faces

The dark blue line indicates the border of the texture, and the green lines indicate the borders of the edges of our book. To render a texture, open the Tools menu and select the Render UVW Template option. In the window that opens, you can set the colors for the image and the size of the created texture.

Set the resolution to 512 pixels and visualize the image by clicking the Render UV Template button.

Save the resulting image to your hard disk, then open it using any image editor, such as Adobe PhotoShop. Let's label each face and save the texture image to the hard disk.

Open the material editor and assign this image to the new material as the Diffuse map. The resulting material will look like this.

Now let's create another book by making a copy of it and assign them this material. Turn the second book over to see its back side.

Now we need to work with our material again in the image editor. But this time we need to place the image of the book on the edges so that the texture scan looks like this.

Save the new texture, and set it as a Diffuse map of the new material. Then create another book using the copy method and assign this material to all the books. Add a plane to the scene that simulates the floor and light sources as desired. Visualize the resulting scene.
